# npm package starter

A boilerplate to help you get started creating a npm package.

## includes

* Rollup
* Babel
* Prettier
* Jest

## How to use

```
git clone git@github.com:gvergnaud/npm-package-starter.git my-package
cd my-package
npm i
npm test
```

### Add your own remote
```
git remote rename origin starter
git remote add origin <your-remote>
```
